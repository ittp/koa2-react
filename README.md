koa2-react
===
A porject with koa2,react,mobx on server side render
***
Install with:

    npm install koa2-react

Usage Example
===
Init

    npm install

Build fe

    npm run fe

Build rd

    npm run rd

Run develop

    npm run dev

Build release

    npm run build

Publish

    npm run release

Run server online

    npm start

Stop server online

    npm stop


Entry
===
config file path:`client/entry.js`
